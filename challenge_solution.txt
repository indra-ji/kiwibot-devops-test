***BUILD PROCESS***

ACCESS 
1. Check whether you have received access to the project on GCP.

SETTING UP GCP 

1. Install and initialize the GCP CLI 
    https://cloud.google.com/sdk/docs/install

2. Set this project as default 
    gcloud config set project kiwibot-devops

3. Configure gcloud for your chosen region 
    gcloud config set run/region us-central1

DEPLOYING INTO PRODUCTION 

1. Creating a Memorystore instance for Redis 
    gcloud redis instances create kiwibot-redis --size=1 --region=us-central1

2. Start a Serverless VPC Access Connector 
    gcloud compute networks vpc-access connectors create connector-kiwibot \
  --region us-central1 \
  --range "10.8.0.0/28"

3. Define environment variable with the IP address of your Redis instance's 
authorized network. 
     export REDISHOST=$(gcloud redis instances describe kiwibot-redis --region us-central1 --format "value(host)")

4. Create service account for service identity.
    gcloud iam service-accounts create chat-identity

5. Build and deploy. 
    gcloud beta run deploy chat-app --source . \
    --vpc-connector connector-kiwibot \
    --allow-unauthenticated \
    --timeout 3600 \
    --service-account chat-identity \
    --update-env-vars REDISHOST=$REDISHOST

    Note: Press Y on any API installs that may be necessary 

USER TESTING 

1. Go to generated URL via your browser. 

2. Add name and name of the room. 

3. Use the application.
